Projects

Here are some of the projects I have had the opportunity to work on. For any enquiries, please contact me on Issabqain[dot]hotmail[dot]com.

Quantized GNN Training

Training Graph Neural networks is an expensive operation which requires increasingly more compute and memory as the input graph size increases. This project focuses on the quantized training of GNNs to tackle this problem.

Electronic Charity Box

During a Christmas charity fundraising run with Greenford Rotary Club, I noticed that an excessive amount of time was taken up to count coin donations. Therefore, to allow fundraisers to spend more time fundraising rather than counting coins I built from scratch a protoype Charity box to reduce time taken to count coin donations

Desinging an Operational Amplifier

Designed an operational amplifier using LT072 and MC33078 Chips to meet rigorous criteria

Signal Detecting Rover

Designed a wirelessly controlled rover with the ability to sense, store and decode data for Acoustic, Infrared and Radio signals carrying UART data. As the project manager, I also distributed tasks amongst the team, set deadlines, maintained documentation and edited the technical report
